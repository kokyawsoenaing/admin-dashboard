## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/kokyawsoenaing/admin-dashboard.git
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run Development Server

```bash
npm run dev
```

---

## 🏗️ Build for Production

```bash
npm run build
```
